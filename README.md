
# LyricsIsland

**ClassWidgets の 插件。可以在主界面显示歌词。**<br/>
本插件将会在127.0.0.1:50063上监听，请确保不要占用该端口。
***
### 使用方法
1. 安装网易云音乐
2. 下载并安装[BetterNCM](https://microblock.cc/betterncm)
3. 打开BetterNCM插件广场，下载并安装LyricsIsland插件
4. 点击重载即可使用
